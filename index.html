Projects

×

Stock Emulator

A Python and Flask web application that simulates buying and selling stocks. Users create an account and start out with $10,000. User information and transaction history are stored in a MySQL database.

Website Repository

Weather Messager

A Python and Flask web application that contacts the Dark Sky API for weather forecasts and parses the JSON response for relevant information and sends an SMS message using Twilio's API to the user's mobile phone. Supports changing cities by zip code by querying a PostgreSQL database for latitude and longitude and updating the coordinates that will be sent to the Dark Sky API.

Repository

Punk Beer

A static web application that uses AJAX requests to the Punk API and parses the JSON response for beer title, image, tagline, and description. There is an option to make requests to the API by ABV percentage. Written in JavaScript, jQuery, HTML5, and Bootstrap.

Website Repository

Unbeatable Tic Tac Toe

An implementation of the classic game Tic Tac Toe played on the command line. Written in Java and uses an artificial intelligence algorithm called the minimax algorithm so the computer can determine its best move. The computer will never lose; the worst-case is a draw.

Repository
